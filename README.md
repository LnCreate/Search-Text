# Search-Text
一个用来帮助我的考研朋友在某文件夹下的所有文件中搜索指定关键词的python代码

## 文件项目
```text
D:\GIT_PROJECT\SEARCH-TEXT
│  app_logic.py
│  main.py
│  Ui_search.py
│
├─dist
│      main.exe
│
└─UI
      search.ui
```
其中, app_logic.py为代码运行逻辑
Ui_search.py为QT5生成的设计界面
main.py为主程序，将设计界面和代码进行关联绑定
dist\main.exe为pyinstaller生成的可执行文件
